2. STS 환경세팅
(1) STS 플러그인 설치
-- Eclipse Market > Java and Web ... jsp 설치

(2) 글자 인코딩 설정
- Window > Preferences 설정창을 띄운다.
-- General > Workspace > Text file encoding을 Other 선택 > UTF-8로 변경
-- Web > CSS Files 에서 인코딩 ISO 10646/Unicode(UTF-8) 선택
-- Web > HTML Files 에서 인코딩 ISO 10646/Unicode(UTF-8) 선택
-- (*이건 안해도 됨) Web > JSP Files 에서 인코딩 ISO 10646/Unicode(UTF-8) 선택

v v v New Project 시, github 클론은 여기까지 v v v
